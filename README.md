# jendobson.greenbutton
Python tools for working with [Green Button](http://www.greenbuttondata.org "The Green Button") data.

To Use:

```python
# Import the script greenbutton.py
import greenbutton.greenbutton as gb

# View the dataframe
gb.df
```
